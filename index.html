Politique de Confidentialité

Dernière mise à jour : 08/02/2026

Important : Simply Libéral+ est une application professionnelle destinée aux infirmiers diplômés d'État exerçant en libéral (IDEL). Cette application traite des données de santé à caractère personnel dans le cadre strict de l'exercice professionnel.

1. Responsable du traitement des données

Simply Libéral
Email : contact@simplyliberal.fr

2. Données collectées et finalités

2.1 Données du professionnel de santé

  • Identité : Nom, prénom, adresse email, numéro de téléphone
  • Données professionnelles : Cabinet, horaires, zone d'intervention
  • Données de connexion : Logs de connexion, adresse IP
  • Données de paiement : Traitées par Stripe (certifié PCI-DSS)

Finalités : Gestion de votre compte, fourniture du service, facturation

2.2 Données des patients

  • Identité : Nom, prénom, date de naissance, adresse
  • Données de santé : Actes de soins, observations, traitements
  • Données de localisation : Adresse pour planification des tournées

Finalités : Planification et suivi des soins infirmiers

Base légale : Article 9.2.h du RGPD (soins de santé). Vous êtes responsable du traitement des données de vos patients en tant que professionnel de santé.

3. Hébergement des données

  • Hébergeur : Scalingo (certifié HDS)
  • Localisation : France / Union Européenne
  • Sécurité : Chiffrement HTTPS/TLS et AES-256

4. Durée de conservation

  • Données du professionnel : Durée de l'abonnement + 1 an
  • Données des patients : 20 ans (obligation légale)
  • Logs : 12 mois maximum

5. Sécurité des données

  • ✅ Chiffrement HTTPS/TLS 1.3
  • ✅ Chiffrement base de données (AES-256)
  • ✅ Authentification JWT sécurisée
  • ✅ Hébergement HDS certifié
  • ✅ Sauvegardes quotidiennes chiffrées

6. Partage avec des tiers

  • Stripe : Paiements (PCI-DSS)
  • Scalingo : Hébergement HDS
  • Google Maps : Géolocalisation (adresses uniquement)

Aucune donnée de santé n'est partagée à des fins commerciales.

8. Cookies

Aucun cookie publicitaire. Uniquement :

  • Token d'authentification (session)

9. Géolocalisation

Collecte GPS uniquement pour :

  • Calcul d'itinéraires de tournées
  • Enregistrement des kilomètres (déclaration fiscale)

Désactivable à tout moment. Non partagée avec des tiers.

10. Mineurs

Application réservée aux professionnels de santé majeurs (18+).
